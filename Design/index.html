Ski Resort Conditions App

This web application allows a user to search weather forecasts and snow conditions at specified resort. The search also displays an imbeded google map with a location marker and a current twitter feed of the specified resort's twitter handle. All of this information is displayed on the page in an easy to read dashboard format. The styling of the page was done using Google's Materialize CSS framework. In it's current state the application is limited by the subscription tier used with the Weather Unlocked API to a single search, but the app is designed to be scalable with subscription tiers. Forecast information is based on 1:00pm local time for the present day at the resort.

Earthquake Tracker

Part of a team that created this slick looking app. Epicenter earthquake tracker allows a user to search for earthquake data for a specific location based on latitude, longitude magnitude and radius. The search then displays a google map showing the location of the 12 most recent earthquake events along with a card showing data for each event. This information is displayed on a dashboard that is specific to the user. If a user signs out and signs back in the last performed search will be used as the new "default" search. The application was built with a React.js front end and an Express server backend. Information on earthquakes was pulled from the USGS Earthquake API and updates in real time. The styling of the page was done using bootstrap. The application was built with a React.js front end and an Express server backend.
